# IOS-Swift-CoreLocationMapKit02  
``` Level : Intermediate ```

In this tutorial we learn how to fetch device GPS  Sensor Data ( Latitude, Longitude). And in other ViewController show it over the Map. This tutorial is made with Xcode 9 and built for iOS 11.

## About

Sample app to fetch device GPS Sensor Data (MapKit, Core Location) 

## Requirements

- iOS 12.0+
- Xcode 10.0+
- Swift 4.0+
- iPhone X+

## Usage

You won't need to Download any extra libaray just ```import UIKit``` inside the project
